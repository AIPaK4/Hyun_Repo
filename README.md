### 1. 에너지 가격 변동이 모빌리티 시장에 미치는 영향 분석

[SKN29_Proj1](https://github.com/AIPaK4/SKN29_Proj1)

| 항목 | 내용 |
|------|------|
| 기간 | 2026.03.30 ~ 2026.03.31 (2일) |
| 기술 스택 | Python 3.10+, MySQL, Streamlit, Plotly |

#### 프로젝트 개요

국산차 월별 등록 데이터(국토교통 통계누리)와 전국 유가 데이터(한국석유공사 오피넷)를 크롤링하여 MySQL DB로 통합하고, 유가 변동과 차종별 등록 대수 간의 시계열 상관관계를 분석했습니다.

#### 핵심 분석 결과

**① 신차 등록의 3개월 물리적 시차**
유가 급변 후 실제 판매 등록까지 평균 3개월의 시차가 존재합니다. 구매 결정에서 출고까지의 행정 소요 기간이 반영된 결과로, 경유차·화물차·승합차에서 가장 두드러지게 나타났습니다.

**② 시장 목적별 유가 반응 민감도 격차**
- 민감군 (상용/물류): 화물차·경유차는 유가 변동에 강하게 반응하며, 고유가가 5~6개월 지속되면 등록 대수가 급감합니다.
- 둔감군 (개인용): 비사업용 승용차는 유가보다 소득·금리 등 거시경제 변수에 더 영향을 받습니다.

**③ 친환경차의 유가 독립성**
전기차·하이브리드 차량은 유가 변동과 유의미한 상관관계가 없으며, 정부 보조금 규모·충전 인프라 보급률·신차 출시 라인업 등 정책 요인이 더 결정적인 영향을 미치는 것으로 확인됐습니다.

#### 주요 기능

- 현대·기아·제네시스·KGM 4개 브랜드 데이터 자동 크롤링 및 MySQL 적재
- 차종 선택 시 등록 대수와 유가 추이를 한눈에 비교하는 Streamlit + Plotly 대시보드

### 2. SKY CASTER — 항공편 도착 지연 예측 시스템

[SKN29_Proj2](https://github.com/AIPaK4/SKN29_Proj2)

| 항목 | 내용 |
|------|------|
| 기간 | 2026.04.30 ~ 2026.05.04 (2일) |
| 기술 스택 | Python 3.11, FastAPI, Streamlit, XGBoost, Scikit-learn, MySQL |

#### 프로젝트 개요

항공편 출발 전 확정된 스케줄 정보와 운항 당일 기상·운영 데이터를 활용해 도착 지연 여부(15분 이상)를 사전에 예측하는 이진 분류 서비스입니다.

#### 데이터셋

| 항목 | 내용 |
|------|------|
| 출처 | Kaggle / BTS TranStats |
| 수집 기간 | 2018.01 ~ 2022.07 |
| 전체 데이터 | 약 2,889만 건 |
| 대상 공항 | 377개 |
| 지연 비율 | 약 17.5% (15분 이상 도착 지연 기준) |
| Train/Test 분리 | 시간 기준 6.5 : 3.5 |

#### 전처리 핵심 프로세스

- 결항·회항 제거 및 운항 후 확인 가능한 컬럼(TaxiOut, AirTime 등) 제거로 데이터 누수 차단
- 시간 순환 인코딩(sin/cos), 강수·적설 이진 피처 등 파생변수 생성
- 공항 혼잡도·노선 평균 비행시간 등 집계 피처 활용
- VIF 분석으로 다중공선성 피처 12개 제거
- 원본 61개 피처 → 파생 확장 79개 → 최종 38개 피처로 정리

#### 모델 성능 비교

| 모델 | ROC-AUC | 지연 Recall | 특징 |
|------|---------|-----------|------|
| **XGBoost** (최종) | 0.8437 | 0.39 | 종합 성능 최우수 |
| Stacking | 0.8412 | 0.27 | 지연 Precision 0.61로 오경보 최소화에 유리 |
| LightGBM | 0.8376 | 0.39 | 경량화 대안 |
| RandomForest | 0.8278 | 0.50 | 미탐지 최소화에 유리 |
| FCNN | 0.6844 | 0.76 | Recall 최고, 오경보 다수 |
